# Task 1: Data Cleaning and Preprocessing

## Overview

Data cleaning is a critical step in the data analytics process. This project focuses on identifying and handling data quality issues such as missing values, duplicate records, and data type inconsistencies to prepare the dataset for further analysis.

## Objectives

* Identify and handle missing values.
* Detect duplicate records.
* Validate data types.
* Improve dataset quality and consistency.
* Prepare the dataset for analysis and visualization.

## Tools Used

* Python
* Pandas
* Jupyter Notebook
* Microsoft Excel

## Steps Performed

1. Loaded the dataset using Pandas.
2. Inspected the dataset structure and information.
3. Analyzed missing values.
4. Checked for duplicate records.
5. Verified data types.
6. Performed necessary data cleaning operations.
7. Validated the cleaned dataset.

## Results

* Missing values were successfully handled.
* No duplicate records were found in the dataset.
* Data types were validated and verified.
* Dataset quality and consistency were improved.
* The cleaned dataset is ready for further analysis and visualization.

## Project Files

* `task1.ipynb` – Jupyter Notebook containing the implementation.
* `Task1_Documentation.pdf` – Detailed project documentation.
* `screenshots/` – Screenshots of outputs and results.

## Conclusion

The dataset was successfully cleaned and preprocessed using Python and Pandas. The resulting dataset is accurate, consistent, and suitable for data analytics applications.
