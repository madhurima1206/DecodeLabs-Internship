# Exploratory Data Analysis (EDA) on Dataset for Data Analytics

##  Project Description

This project performs Exploratory Data Analysis (EDA) on a cleaned dataset to understand its structure, identify trends, analyze sales patterns, detect outliers, and derive meaningful insights. The analysis was conducted using Python and Pandas in Jupyter Notebook.

##  Objectives

- Understand the dataset structure and characteristics.
- Perform descriptive statistical analysis.
- Analyze product sales performance.
- Examine daily and monthly sales trends.
- Study order status distribution.
- Analyze payment method preferences.
- Detect potential outliers.
- Extract useful business insights.

##  Dataset Information

| Property | Details |
|----------|----------|
| Dataset Name | Cleaned Dataset for Data Analytics |
| Format | Excel (.xlsx) |
| Domain | Data Analytics |
| Purpose | Exploratory Data Analysis |

##  Tools & Technologies

- Python
- Pandas

##  Analysis Performed

### 1. Data Loading
- Imported the cleaned dataset using Pandas.

### 2. Data Inspection
- Dataset shape analysis
- Dataset information review
- Data type verification

### 3. Descriptive Statistics
- Mean
- Median
- Minimum values
- Maximum values
- Data distribution analysis

### 4. Product Sales Analysis
- Total quantity sold per product
- Most sold products
- Least sold products

### 5. Daily Sales Analysis
- Daily sales calculation
- Daily sales trend analysis

### 6. Monthly Sales Analysis
- Monthly sales calculation
- Highest sales month identification
- Lowest sales month identification

### 7. Monthly Order Analysis
- Monthly order count
- Monthly order trend analysis

### 8. Order Status Analysis
- Order status distribution
- Most common order status

### 9. Payment Method Analysis
- Payment method usage frequency
- Most preferred payment method

### 10. Outlier Detection
- Identification of unusual values
- Impact analysis of outliers

##  Key Findings

- Product sales varied significantly across products.
- Daily and monthly sales patterns were identified.
- Order volumes fluctuated across different months.
- Some payment methods were preferred more than others.
- Statistical analysis provided valuable business insights.
- Potential outliers were detected in numerical columns.

The analysis successfully revealed:

- Daily sales patterns
- Monthly sales trends

- Insights useful for decision-making
